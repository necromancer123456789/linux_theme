Note:
.bashrc file is only loaded when you open a terminal, if you want to add an startup program then you will need to use systemd instead.

1. goto the directory where the file termnalTheme.sh is located.
2. right click then choose open in terminal.
3. then run this command:

bash ./termnalTheme.sh

Note:
if the theme does not automatically apply then you will need to manually apply it, to your settings.

or

open a terminal and run this command:

source ~/.bashrc
